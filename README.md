VR Nature Trail – Forest Maze (Unity Project)

Project Overview
This project presents a virtual environment developed using the Unity game engine, designed to simulate a guided nature exploration experience. The system allows users to navigate through a structured maze-like terrain using first-person controls.

The environment focuses on accessibility, smooth navigation, and controlled interaction within a nature-inspired setting.

---

Features
- First-person navigation using keyboard and mouse  
- Maze-based environment for guided exploration  
- Terrain creation with textured walkable paths  
- Hedge-like wall structures for movement boundaries  
- Collision detection for restricted navigation  
- Environmental elements such as grass and water bodies  
- Lightweight and stable real-time performance  

---

Controls

Action | Key
------ | -----
Move Forward | W
Move Backward | S
Move Left | A
Move Right | D
Look Around | Mouse

---

How to Run
1. Download or clone the repository  
2. Open the Build folder  
3. Run:
   ForestMaze.exe  
4. Ensure all files in the folder are kept together
